# Stoke AI logo assets

Saved from Telegram uploads and normalized for website/checkout use.

Recommended defaults:

- `stoke-ai-logo-wide-transparent.png` — main wide logo on light or white backgrounds.
- `stoke-ai-logo-stripe-wide-transparent.png` — upload as the Stripe checkout/logo image if Stripe accepts a wide logo.
- `stoke-ai-icon-stripe-white.png` — upload as the Stripe square icon/avatar if needed.
- `stoke-ai-icon-square-transparent.png` — transparent square flame icon for favicons/app icons or dark/light layouts.
